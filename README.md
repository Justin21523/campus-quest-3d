# Campus Quest 3D

A full-stack 3D campus adventure game platform.

## Tech Stack
- **Frontend**: React, TypeScript, Three.js, React Three Fiber, Tailwind CSS, Phaser.js
- **Backend**: Node.js, Fastify, TypeScript, MongoDB
- **AI Service**: Python, FastAPI, Pydantic

## Getting Started

### Prerequisites
- Node.js >= 18
- Python >= 3.10
- npm >= 9

### Installation
```bash
npm install
cd apps/ai-service && pip install -r requirements.txt && cd ../../
